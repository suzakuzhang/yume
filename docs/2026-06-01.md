## 1. 技术栈 / 架构

- **Next.js 14 (App Router) + TypeScript + Tailwind 4**，从 `zhouyi` 克隆外壳。
- **存储 = JSON-on-disk**（`src/lib/store/`，落 `${DATA_DIR}/yume_data.json`）。
  本机 Node 26 编译不了 `better-sqlite3`，故用纯 JS JSON store；SQLite 实现留在
  `src/lib/store/db.sqlite.bak`，生产（Render 稳定 Node）可切回。store 是唯一调用面。
- **LLM = aihubmix 单网关**（`src/lib/llm/callLLM.ts`，OpenAI 兼容 `/chat/completions`）。
  一个 `AIHUBMIX_API_KEY` 覆盖 DeepSeek/Gemini/… 聊天 + 生图。
- **起卦/解卦引擎、64 卦数据、八卦属性**：原样移植自 zhouyi（`src/lib/casting`、`reading`、`data`）。
- **塔罗**：22 张大阿卡纳移植自 `tarot_local_test`（`src/data/tarot_cards.json` + `public/tarot/cards/`）。
- **知识库**：`yume/knowledge_base/`（7 领域 dossier + `symbol_dictionary.json` + GAPS），
  运行时用 `src/data/symbol_dictionary.json` 做符号检索。

## 2. 设计调性（已立）

- **「阈 / the threshold」**：催眠态、墨(周公周易)·星(塔罗占星)·渊(精神分析)。`docs/tone.md`。
- **导航 = 点击坠落的牌组，全程无滚动**；滚动条全局隐藏（`globals.css`）。
  坠落动效 `yume-plunge` + `yume-blackout` + `yume-surface`（~0.66s，加速下坠→黑场→从黑中浮起）。
- **当日元素驱动 UI**：`--element` CSS 变量（zh=日主五行色 / en=太阳星座四元素色），
  元素文字呼吸发光（`.elem-glow`），坠落按钮 = 当日元素**手绘符号**（火焰/水滴/风/刃/山，浮雕微光 `.elem-orb`）。
- **字体**：宋体优先（Songti SC…）。**全文案无破折号**。
- 背景：公领域古星图（en=Cellarius、zh=苏州天文图拓片）虚化旋转 + 极淡月相盘。

## 3. 落地牌组（`src/app/page.tsx`）

`language → threshold → compass → close`，每屏点击专属按钮坠入下一屏：
1. **language**：夢/入梦 │ dream/drift in（点一侧=定语言+坠入）。**永远第一屏**（无阻塞式 gate）。
2. **threshold**：`yume 夢` + 招魂句 + 发光的「今日丙午·火 ☾ 时分」(en: ☉ Gemini · Air) + **「入梦」**按钮。
3. **compass（`GazeCompass.tsx`）**：四目光罗盘，指针随悬停转向，圆心显影头像+名字，下方该位描写+名言。
4. **close**：「认识你自己 / Know thyself」(金, 原大小) + 仪式弧 + 此刻面板(含时区) + 入口按钮。

## 5. 梦境解读（`src/app/dream/[id]/page.tsx`）= 仪式翻页

`入夜 → 凝象 → 问卜 → 众声 → 拂晓`，进入某屏自动跑该阶段、可点元素符号坠入下一屏：
- **入夜**：梦的意象/问题/心绪/今日基音。
- **凝象**：意象生图（`lib/painterly`；无 key 给笔墨描写，有 key 出图）。
- **问卜**：自动起卦(梦象起卦 seed)+抽塔罗。
- **众声**：自动跑四目光辩论（`lib/debate/engine.ts`：无 key=静态象义；有 key=立论→求同存异→综合）。
- **拂晓**：综合（求同/存异/建议/自我探寻）+ 认识你自己。
