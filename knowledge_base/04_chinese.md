# 中国传统占梦 / Chinese oneiromancy (周公解梦·梦林玄解·王符·周易)

中国传统占梦是一套绵延三千年的释梦体系，核心信念是「梦为兆」——梦境是天人、神形、阴阳之气交感时漏出的征象，可据以预测吉凶。它沿两条互补的脉络发展。**官方/经学脉**：《周礼·春官》设「占梦」官，「以日月星辰占六梦之吉凶」（正梦、噩梦、思梦、寤梦、喜梦、惧梦），把梦纳入与龟卜、筮占鼎足而三的国家占卜制度；《列子·周穆王》以「神遇为梦，形接为事」「觉有八征，梦有六候」给出心理-生理的成梦机制。**民间/术数脉**：托名周公旦的《周公解梦》（实成于汉魏、敦煌存唐写本 P.3908《新集周公解梦书》）建立了庞大的「符号→吉凶」查表，按天文、地理、人身、器物、衣冠等分门，逐条断验（如「齿落更生子孙兴」「上高堂大富贵至」）；明代陈士元《梦占逸旨》《梦林玄解》是其集大成的类书。

**它如何解一个梦**：传统占梦不止机械查表，更讲三层判断。其一，**辨梦之真伪与类型**——王符《潜夫论·梦列》立十类梦（直、象、精、想、人、感、时、反、病、性），先判此梦属「直应之梦」（直接预兆）还是「想梦」（日有所思）、「病梦」（脏腑致梦，呼应《黄帝内经·灵枢·淫邪发梦》「阴气盛则梦涉大水，阳气盛则梦大火」）或「反梦」（阴极反吉、阳极反凶）。其二，**因人而异**——同一象「贵人梦之即为祥，贱人梦之即为妖」，须结合做梦者身份、时令（「春梦发生，秋冬梦熟藏」）。其三，**取象断吉凶**——以谐音、类比、阴阳五行推演（蛇/龙主贵子、火主兴旺发财）。

**与周易/占卜的关系**：占梦与《周易》筮占同源于「观象-取义-断占」的思维。梦象可视作未起之卦象，故民间有「以梦象起卦、再以卦爻辞断梦」的接法（梅花易数式即景起卦传统）；亦有「梦有五不占」的限定（无诚、无据之梦不占）。对 yume 而言，占梦透镜提供的是「象数符号库 + 吉凶映射 + 取象逻辑」，可与起卦模块共享一套阴阳五行/类象的语言。

## 经典著作 / Canonical Works

### 周礼·春官宗伯（占梦）

- **作者**: 传周公旦，实成于战国，西汉刘歆校定  **年份**: 约公元前3世纪成书
- **核心概念**: 占梦官职, 六梦：正/噩/思/寤/喜/惧, 以日月星辰占六梦之吉凶, 季冬献吉梦于王、赠恶梦, 与卜、筮鼎足
- **为何重要**: 中国占梦最早的官方制度记载与最权威的六梦分类，确立梦在国家占卜体系中的正统地位
- **合法全文**: 🟢 公共领域 — 维基文库《周礼·春官宗伯》 https://zh.wikisource.org/wiki/%E5%91%A8%E7%A6%AE/%E6%98%A5%E5%AE%98%E5%AE%97%E4%BC%AF

### 潜夫论·梦列

- **作者**: 王符（东汉）  **年份**: 约公元2世纪
- **核心概念**: 十类梦：直/象/精/想/人/感/时/反/病/性, 直应之梦, 象梦取譬, 贵贱异占, 反梦（阴极吉、阳极凶）
- **为何重要**: 中国占梦理论的高峰，第一次系统区分梦的成因与类型，把『不是所有梦都是预兆』讲清楚，奠定后世辨梦框架
- **合法全文**: 🟢 公共领域 — 维基文库《潜夫论》 https://zh.wikisource.org/wiki/%E6%BD%9B%E5%A4%AB%E8%AB%96

### 列子·周穆王

- **作者**: 传列御寇，今本魏晋辑成  **年份**: 今本约公元4世纪
- **核心概念**: 神遇为梦，形接为事, 觉有八征，梦有六候, 六候同《周礼》六梦, 昼想夜梦, 古之真人其寝不梦
- **为何重要**: 提出梦的心理-哲学机制，将六梦与『神/形』『想/遇』挂钩，是占梦从巫术走向哲思的关键文本
- **合法全文**: 🟢 公共领域 — 维基文库《列子·周穆王篇》 https://zh.wikisource.org/zh-hans/%E5%88%97%E5%AD%90/%E5%91%A8%E7%A9%86%E7%8E%8B%E7%AF%87

### 庄子·齐物论（庄周梦蝶）

- **作者**: 庄周及后学  **年份**: 约公元前3世纪
- **核心概念**: 庄周梦蝶, 不知周之梦为胡蝶，胡蝶之梦为周, 物化, 梦觉之辨, 齐物
- **为何重要**: 中国梦哲学的根脉，对『梦与醒、真与幻』提出根本质疑，与占梦的『梦必有兆』形成张力，是哲学透镜与术数透镜的分水岭

## 核心概念 / Key Concepts

- **六梦（正/噩/思/寤/喜/惧）** — 《周礼·春官》占梦官以日月星辰所占的六类梦：正梦（无所感动平静而梦）、噩梦（惊愕恐怖之梦）、思梦（有所思而梦）、寤梦（觉时所道、半醒之梦）、喜梦（喜悦而梦）、惧梦（恐惧而梦）
  - *用于解梦*: 作为顶层情绪/状态分类器：先把用户的梦归入六梦之一，决定后续是『取兆占断』（正梦/噩梦）还是『归因心理』（思梦/喜梦/惧梦）
- **十梦说（直/象/精/想/人/感/时/反/病/性）** — 王符《潜夫论·梦列》按成因分十类：直梦（直接应验）、象梦（取象譬喻）、精梦（精诚所感）、想梦（昼思夜梦）、人梦（因人身份吉凶不同）、感梦（风雨寒暑气候所感）、时梦（四时之气）、反梦（阴极吉阳极凶）、病梦（疾病所致）、性梦（秉性好恶所致）
  - *用于解梦*: 占梦透镜的核心推理引擎：在断吉凶前先判梦的『类型』，避免把『想梦/病梦』误当作预兆，体现『不是所有梦都该占』的克制
- **直梦 / 直应之梦** — 梦中所见与现实将发生之事直接对应、无需转译的预兆梦
  - *用于解梦*: 对应可直接查表断吉凶的梦；占梦 agent 在判定为直梦时才给出明确预言式回答
- **象梦 / 取象** — 梦以譬喻、象征出现，如《诗经》『维熊维罴，男子之祥；维虺维蛇，女子之祥』，须解码象征义
  - *用于解梦*: 象征解码层：用类象/谐音规则把意象译为吉凶（如蛇→女子之祥/生贵子），是与荣格象征法表面相似的接口
- **反梦（极反）** — 阴极则吉、阳极则凶——某些梦的吉凶与表面相反，如梦凶反主吉
  - *用于解梦*: 为『噩梦未必凶、吉梦未必吉』提供传统依据；agent 可对极端情绪梦给出反向解读，制造与直觉冲突的洞见
- **病梦** — 因脏腑气血失调而生的梦，《灵枢·淫邪发梦》详列『阴盛梦涉大水、阳盛梦大火、上盛梦飞、下盛梦堕』
  - *用于解梦*: 身体信号维度：把反复出现的水/火/飞/坠梦提示为身心状态而非预兆，与现代睡眠/情绪解读自然衔接
- **神遇为梦，形接为事** — 《列子·周穆王》：梦是『神』（精神）所交、事是『形』（身体）所接，昼想夜梦皆神形所遇
  - *用于解梦*: 成梦机制的本土表述，可作 app 向用户解释『为何会做这个梦』的传统话术
- **庄周梦蝶 / 物化** — 《庄子·齐物论》以梦蝶寓言质疑梦觉、物我之分界，归于『物化』（万物互化、彼我两忘）
  - *用于解梦*: 提供反占卜、反执着的哲学声部：当用户焦虑于『梦是否成真』时，引入『梦觉一如、不必执于吉凶』的疏导
- **众占非一，而梦为大** — 陈士元《梦占逸旨》语，强调在诸般占卜中梦占最为重要
  - *用于解梦*: 占梦透镜的自我定位宣言，可用于 agent 的身份/语气设定
- **梦有五不占，占有五不验** — 传统占梦原则：某些梦（如不诚、无据、神乱之梦）不当占，某些情形占而不验
  - *用于解梦*: 为 agent 设置『拒答/降低置信度』的边界，避免对每个梦都强行断吉凶
- **贵贱异占（因人而异）** — 王符『贵人梦之即为祥，贱人梦之即为妖』——同象因做梦者身份而吉凶相反
  - *用于解梦*: 情境化解读：结合用户处境/身份调整同一意象的吉凶，而非一刀切查表
- **时梦（四时之气）** — 『春梦发生，夏梦高明，秋冬梦熟藏』——梦随季节之气而变
  - *用于解梦*: 时间情境因子：可让占梦解读纳入季节，呼应起卦模块的时令变量
- **梦禳（禳解恶梦）** — 《梦林玄解》专列『梦禳』，记载化解噩梦的禳祝之法；《周礼》占梦官季冬『赠恶梦』即此
  - *用于解梦*: 为噩梦提供『化解仪式/安心建议』的传统出口，而非只报凶
- **梦象起卦（梦与周易接口）** — 把梦中意象视作未显之卦象，以象数（阴阳五行、八卦类象）起卦并以卦爻辞断梦的接法
  - *用于解梦*: 直接对接 yume 起卦模块：梦意象→八卦类象→起本卦/变卦→爻辞断验，共享一套类象词典

## 符号映射 / Symbol Mappings

- **火**: 总体主兴旺发财：『火烧山野大显达，火焰炎炎主发财，火烧自是主兴旺，大火烧天主国安，身在火中贵人扶』；但『火从地生疾病至，火烟黑色主疾病』为凶。《灵枢》另解为阳气盛之病梦
- **盔甲 / 甲胄**: 『甲胄披身护吉利』——披挂铠甲主有护佑、得保护，吉象
- **水**: 多吉：『水上行者主大吉，水上火出主大吉，大水澄清大吉祥，自在水中大吉利』；唯『水上立者主凶事』。《灵枢》以梦涉大水为阴气盛/恐惧之病梦
- **蛇**: 多主财与贵子：『蛇咬人主得大财，蛇入怀中生贵子，蛇绕身者生贵子，龙蛇入门主得财』；女子梦蛇为生女之祥（象梦『维虺维蛇，女子之祥』）
- **牙齿脱落**: 区分主体：『齿自落者父母凶』（主父母之忧）；但『齿落更生子孙兴』（落而复生反主子孙兴旺）
- **坠落 / 堕**: 《灵枢·淫邪发梦》『下盛则梦堕』——身体下部气盛、阴阳失衡所致的病梦信号；传统亦视坠落为失位、厄运之兆
- **飞 / 飞行**: 《灵枢》『上盛则梦飞』——身体上部气盛之病梦；民间亦有飞升主升迁显达之吉解
- **棺 / 棺材**: 谐音『官/财』多主吉：『将棺入宅禄位至，棺敛死人主得财』；但『开棺与死人言凶』为凶象
- **血**: 见血多主得财酒食：『杀人血污衣得财，刀伤出血主酒食，砍刺见血主大吉』
- **龙**: 大贵之象：『龙飞有官位大贵，乘龙入水有贵位，龙蛇入门主得财，妇人见龙生贵子，乘龙上山所求遂』
- **登高 / 上山上楼**: 主富贵高升：『上楼阁坛俱大吉，上高堂大富贵至，乘龙上山所求遂』
- **哭 / 哭泣**: 反兆为主：『与人哭泣有庆贺，放声大哭欢乐生』（哭反主喜）；唯『床上哭泣主大凶』
- **镜 / 照镜**: 以明暗断吉凶：『照镜明吉暗者凶』；又『拾得镜者招好妻，将镜自照远信至』
- **刀剑**: 多主吉利得财：『拔刀出行大吉利，剑在床头大吉利，持刀相杀见血吉』；唯『持刀刺人主失利』
- **桥**: 造修桥主和合禄位：『修桥梁者万事和，桥上坐立禄位至，新造桥者大和合』；另说『见渡桥者主厄散』（厄运消散）

## 作为辩论 Agent 的人设 / App Integration

声音：一位援引经典、讲究『辨类而后断』的占梦师——先问做梦者身份、时令、近日心境，再判此梦属六梦/十梦中的哪一类，最后才取象断吉凶。语气克制权威，常引『众占非一，而梦为大』，也常提醒『梦有五不占』。强调：(1)符号→吉凶的明确映射（火主兴旺、棺主官财、蛇主贵子、齿落主父母/子孙、照镜明吉暗凶等），给用户可执行的『吉/凶+建议』；(2)分类先行——把『想梦/病梦』剥离出去不强行占卜；(3)反兆智慧（哭反主喜、噩梦未必凶）制造反直觉洞见；(4)噩梦给『梦禳/化解』出口而非纯报凶。与起卦模块对接：可把梦中主意象映射为八卦类象（火→离、水→坎、龙/雷→震、山/止→艮等）起卦，再以卦爻辞复核梦占，两套象数共用一部类象词典。与其他透镜的冲突：与弗洛伊德最对立——占梦认定梦指向外部未来事件(预兆)，弗洛伊德认定梦指向内部past欲望(愿望满足)，且占梦的象固定查表、弗氏的象因人自由联想；与荣格表面相近（都用象征/原型，如蛇、水），但占梦的象义是文化约定且断吉凶，荣格的象义是个体化的心理整合、不论吉凶；与西方/塔罗占卜同属预测系，可结伴但方法不同（梦占重类象与谐音，塔罗重牌阵）。盲点：(1)宿命论倾向——预设梦必有兆，易忽视梦的心理/生理成因（虽『病梦』已部分弥补）；(2)符号映射高度文化绑定且条目互相矛盾、版本众多，缺乏可证伪性；(3)托名伪书、坊本驳杂，权威性需打折；(4)对创伤、情绪处理无能为力——这是它应主动让位给心理学透镜之处。理想用法：用作『文化象征 + 吉凶取向 + 起卦接口』的一票，而非唯一裁决者。
